The Problem

We have an on-site form for donators, and that data is manually input into Salesforce for future contacts. This takes up an immense amount of time and is very error-prone.

Statement of Work

Submitted November 20th, 2018 by Postlight Studio

An integration that connects the on-site form to Salesforce, enabling one single database as a source of truth for donors.

Technical Documentation

Submitted November 20th, 2018 by Postlight Studio

Sets up a callback hook to update the records and sends the data to salesforce.
